1. Description

This example demonstrates how to use the AD1854 DAC driver and AD1871 ADC driver.

2. Processor

  • ADSP-BF537

3. Tested with

  • ADSP-BF537 EZ-KIT Lite
  • Analog Devices CrossCore Embedded Studio
  • ADDS HPUSB-ICE

4. Overview

This example uses the AD1854 Stereo Audio DAC driver and AD1871 Stereo Audio ADC driver to do audio loopback between ADC and DAC.
The program copies the data received from ADC to a DAC buffer without any modification and submits it to the DAC driver.
 
The AD1854 and AD1871 drivers support both non-blocking and callback modes, and the example uses a macro to demonstrate both modes.

5. Hardware Setup

  1. ADSP-BF537 EZ-KIT Lite switch settings:

    SW7:  1(ON),  2(ON),  3(ON),  4(ON),  5(ON),  6(ON),  7(ON),  8(ON)

    Others - don't care

  2. Connect an audio souce to J9 (Line In) and an audio output source (Headphone or speaker) to J10.

6. Running the Example

  1. Select the operating mode using macros listed in 'AudioLoopback.h' file.

    Default settings: Callback disabled, Run application in an infinite loop, Debug information enabled.

    As an example scenario, following macros should be enabled to operate the example in callback mode with application time-out.

    • Enable ENABLE_CALLBACK macro
    • Enable ENABLE_APP_TIME_OUT macro

  2. Connect an audio souce to J9 (Line In) and an audio output source (Headphone or speaker) to J10.

  3. Build the project, load the executable to ADSP-BF537, and run it (press resume button in debug view or hit F5).

  4. Look for the debug information displayed on Analog Devices CrossCore Embedded Studio console window.

  5. Listen to audio palyed over the audio output source.

7. User Configuration Macros

User configuration macros listed in Table 1 are specified in the include file AudioLoopback.h'.

Table 1: User Configuration macros
Macro
 Function
ENABLE_CALLBACK
Enable macro to build example in callback mode.
Disable to build in non-blocking mode.
ENABLE_APP_TIME_OUT
 Enable macro to terminate the example after the time-out value expires.
Disable macro to run the example in an infinite loop.
ENABLE_DEBUG_INFO Enable macro to display debug information.
